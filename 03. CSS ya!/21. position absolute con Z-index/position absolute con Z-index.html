Hemos visto que cuando disponemos uno o más elementos con la propiedad position con valor absolute los mismos salen del flujo del resto de elementos de la página.

En esta situación debemos inicializar la propiedad z-index con un valor entero, el mismo indica cual tiene prioridad para visualizarse. El valor más grande indica cual tiene prioridad para visualizarse. Es decir que el navegador imprime los elementos con posición absoluta del valor más bajo al más alto.

Con el valor z-index podemos configurar diferentes imagenes o archivos y posicionarlos uno encima de otro, podemos elegir cual tendra prioridad para visualizarse encima que el otro.
