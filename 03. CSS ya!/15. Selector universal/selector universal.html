Selector universal *

Si queremos inicializar las propiedades de todas los elementos HTML podemos utilizar el selector universal. Utilizamos el carácter asterisco para hacer referencia a este selector.

Ejemplo:
* { margin:0; padding:0; color:#ff0000; }

Con dicho selector indicamos a todos los elementos html tendran un margin y padding de cero y los texto de color rojo. Aun asi podemos añadir elementos con otras configuraciones, ejemplo añadir un "h1 { color:#ff0000}", este tendra prioridad
