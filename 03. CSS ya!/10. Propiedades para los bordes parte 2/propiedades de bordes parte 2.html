Como vimos en el concepto anterior tenemos propiedades que nos permiten fijar el grosor, estilo y color del borde de un elemento HTML. Ahora veremos como modificar cada uno de los bordes del rectángulo.

Contamos con las siguientes propiedades:

border-top-width
border-right-width
border-bottom-width
border-left-width


border-top-style
border-right-style
border-bottom-style
border-left-style


border-top-color
border-right-color
border-bottom-color
border-left-color


Resumido:
border-width: 5px 7px 5px 7px;
border-style: groove dotted double dashed;
border-color: red yellow blue green;


Este texto estara encerrado por un border. Este cuadro va hacer usado como ejemplo y vamos a editar cada uno de sus border mediante css. TU WEB Informativa
