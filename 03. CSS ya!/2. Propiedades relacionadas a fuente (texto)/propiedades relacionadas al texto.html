Contamos con una serie de propiedades para personalizar nuestro texto. contamos con:

font-family:
(fuentes mas comunes: Arial
Arial Black
Arial Narrow
Courier New
Georgia
Impact
Tahoma
Times New Roman
Verdana)

En caso que la fuente no esté disponible el navegador selecciona el estilo por defecto para ese elemento HTML.
Podemos definir varias fuentes por si acaso alguna no se encuentra disponible para el navegador, esto lo hacemos separando por coma todas las fuentes (se eligen de izquierda a derecha):

font-size: Indica el tamaño en px para nuestra fuente

font-style: Esta propiedad puede tener los siguientes atributos: normal, italic, y oblique

font-weight: Este propiedad puede tener los siguientes valores:
normal
bold
bolder
lighter
100
200
300
400
500
600
700
800
900

Esta propiedad indica el peso de la fuente (mientras tenga un valor mayor los caracteres serán más rellenos)

font-variant: Este atributo define si se mostraran las letras en mayusculas o minusculas. "small-caps" o "normal"
