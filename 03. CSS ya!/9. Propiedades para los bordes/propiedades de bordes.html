Las propiedades mas importantes de borde son:

border-width. Ancho del borde
border-style. Estilo del borde
border-color. Color del borde.

Este parrafo escrito en html contendra un border, esto permite que el parrafo resalte y tenga un estilo mas animado, ademas contiene un color de fondo (background-color) y un color de texto (letras blancas). Un borde puede ser añadido a una imagen o otros archivos html. TU WEB Informativa
