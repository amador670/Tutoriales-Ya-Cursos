Eventos en JavaScript segun la W3C

"Este metodo es mi favorito"

addEventListener() Registra un evento a un objetivo en especifico. El ObJeto especifico Puede ser un simple elemento en un archivo, el mismo documento , una ventana, o un XMLHttpRequest.

El método document.addEventListener() asigna un controlador de eventos al documento.

Sugerencia: Utilice el método document.removeEventListener() para eliminar un controlador de eventos que se ha adjuntado con el método addEventListener ().

Consejo: utilice el método element.addEventListener() para adjuntar un controlador de eventos a un elemento especificado. 

      Añadir un evento:
      document.addEventListener("mousemove", myFunction);
      Remover un evento:
      document.removeEventListener("mousemove", myFunction);
