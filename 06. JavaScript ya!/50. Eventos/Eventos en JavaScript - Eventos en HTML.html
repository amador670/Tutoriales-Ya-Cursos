Eventos en JavaScript - Eventos en HTML

Mediante este metodo los eventos de JavaScript son llamados en HTML mediante funciones, empleando en la parte de HTML propiedades como "onclick", "onsubmit", entre otras, que permiten la ejecucion de dichas funciones, esta forma es muy rapida y sencilla, ya que no lleva tantos procesos en JavaScript, pero no es la mejor, ya que parte de nuestro codigo HTML va a contener codigo JS y a la hora de modificarlo sera mas el codigo que hay que modificar.
