Propiedad windows location

Esta propiedad nos permite dirigirnos a un enlace mediante un <a href> que va a contener una función en vez de un enlace, añadiendo en la función "window.location" nos indica que la "ventana tendrá otra localización" en este caso un enlace.

Otro ejemplo de window location es:

   function redireccionar() {
    var enlaceWeb = confirm("¿Desea ingresar al sitio web oficial?");

    if(enlaceWeb == true){
     window.location="http://www.tuwebinformativa.net";
    } else if(enlaceWeb == false){
     alert("El acceso a la pagina web oficial fue cancelado");
    }
   }
