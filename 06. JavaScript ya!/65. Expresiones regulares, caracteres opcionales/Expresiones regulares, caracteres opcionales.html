Expresiones regulares, caracteres opcionales

El carácter ^ al principio /^[]/ se usa para especificar que carácter, palabra o numero, sera el ingresado al principio, este se encarga de los primeros caracteres ingresados.

El carácter $ al final /[]$/ se usa para especificar que carácter, palabra o numero, sera el ingresado al final, este se encarga de los últimos caracteres ingresados

Extra:

Para mostrar una alerta o mensaje solo para letras se usa [a-z], si queremos permitir también mayúsculas usamos [a-zA-Z]

Para mostrar una alerta o mensaje solo para números se usa [0-9]

Si queremos que aparezca una alerta o mensaje para los signos y caracteres se usa /[\^$.*+?=!:|\\/()\[\]{}]/

Los caracteres que sean ingresado en secuencia, ejemplo [aeiou] especifican la secuencia que se cumplirá (mediante una alerta o mensaje). Sean al principio o al final estos especifican que al ingresar esos caracteres se ejecutara una determinada acción
