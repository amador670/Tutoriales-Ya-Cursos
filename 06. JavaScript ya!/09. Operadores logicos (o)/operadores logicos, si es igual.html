El operador logico || (o) nos permite añadir varias propiedades y indicar que algunas de ellas se van a cumplir, con este operador indicamos que si no se cumple una funcion se cumple la otra y asi sucesivamente. Ejemplo

if (nombre=="Amador Jose" || apellido=="Martinez Rivera") {
    document.write("Tu eres el creador de la pagina. Bienvendo!!");
} else {
    document.write("Bienvenido a mi sitio web");
}
