Array .sort()

El método sort () ordena los elementos de una matriz.

El orden puede ser tanto alfabético o numérico, ya sea ascendente (arriba) o descendente (abajo).

De forma predeterminada, el método sort () clasifica los valores como cadenas en orden alfabético y ascendente.

Esto funciona bien para cadenas. Sin embargo, si los números se ordenan como cadenas numericas, "25" es mayor que "100", porque "2" es más grande que "1". Debido a esto, el método sort() producirá un resultado incorrecto al ordenar números.

Array .reverse()

El método reverse() coloca al revés (inversamente) una matriz. El primer elemento pasa a ser el último y el último pasa a ser el primero.
