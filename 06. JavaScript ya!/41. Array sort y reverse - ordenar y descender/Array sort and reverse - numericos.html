Array .sort() y .reverse() de forma numerica

Para organizar un array con .sort() de forma numerica se emplea un funcion (a, b) con dos parametros para especificar un orden correcto. Ejemplo:

     Forma ordenada:
      var points = [40, 100, 1, 5, 25, 10];
      points.sort(function(a, b){return a-b});
    
     Forma descendiente:
     var points = [40, 100, 1, 5, 25, 10];
     points.sort(function(a, b){return b-a});
