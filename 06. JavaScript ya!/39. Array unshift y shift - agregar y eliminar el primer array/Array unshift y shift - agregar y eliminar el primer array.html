Metodo .unshift()

Este metodo añade nuevos elementos al principio de un Array

Ejemplo de este metodo:
    var vector = [1, 2];

    vector.unshift(0); // El resultado seria 3 array, el 0 se añade al principio:
    // vector is [0, 1, 2]

    vector.unshift(-2, -1); // El resultado seria 5 array, el -2, -1 se añade al principio:
    // vector is [-2, -1, 0, 1, 2]

    vector.unshift([-3]); // El resultado seria 6 array:
    // vector is [[-3], -2, -1, 0, 1, 2]

Metodo .shift()

Este elemento remueve el primer elemento de un array
