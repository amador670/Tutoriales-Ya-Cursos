Propiedad math.sqrt()

Con esta propiedad podemos sacar la raíz de un determinado numero mediante JavaScript

La propiedad .toFixed() es para los valores numéricos, esta propiedad se encarga de los decimales, el numero que le otorguemos como parámetro .toFixed(2) sera los decimales que mostrara, en este ejemplo dos.

Aproximación, la propiedad .toFixed() aproxima los decimales, ejemplo el valor numérico "5.56789" sera aproximado a "6".

Añade mas decimales, si a un valor numérico se le añade la propiedad .toFixed() con un parámetro mayor a la cifra resultante añade ceros al final, ejemplo 123.10.toFixed(10) añade 10 decimales, este empieza 123.1000000000
