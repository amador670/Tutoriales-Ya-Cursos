Evento focus y blur en JavaScript

Evento focus
El evento de focus ocurre cuando un elemento obtiene el foco.

El evento de focus se suele utilizar con <input>, <select> y <a>.

Sugerencia: El evento focus es el opuesto al evento blur.

Nota extra: El evento de focus es similar al evento focusin. La principal diferencia es que el evento focus no burbujea. Por lo tanto, si desea averiguar si un elemento o su hijo obtiene el foco, puede utilizar el evento focusin. Sin embargo, puede lograr esto utilizando el parámetro opcional useCapture del método addEventListener() para el evento focus.

Evento blur
El evento blur se produce cuando un objeto pierde foco.

El evento blur se usa con más frecuencia con código de validación de formulario (por ejemplo, cuando el usuario deja un campo de formulario).

Consejo: El evento blur es el opuesto al evento focus.

Sugerencia: El evento onblur es similar al evento focusout. La principal diferencia es que el evento onblur no burbujea. Por lo tanto, si desea averiguar si un elemento o su hijo pierde foco, podría utilizar el evento onfocusout. Sin embargo, puede lograr esto utilizando el parámetro opcional useCapture del método addEventListener () para el evento onblur.
