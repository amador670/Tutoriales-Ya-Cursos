Evento onFocus y onBlur

onFocus, esta propiedad se aplica al colocar el cursor en un elemento, se ejecuta cuando un componente de nuestra web toma el foco.

onBlur, esta propiedad se aplica al quitar el cursor de un elemento, este evento es lo contrario, se ejecuta cuando el componente pierde el foco.

En este ejemplo, se creamos dos input type="texto", y haremos que cuando tome el foco (onFocus), el campo de texto borre el texto que tenga escrito, y que cuando pierda el foco (onBlur) el campo de texto, nos lo avise con una ventana JavaScript.
