Una fantastica propiedad que nos brinda CSS3 y que personalmente descrubi por mi persona (alardeo jejeje) es la propiedad de cambiar el color de un parrafo (etc) a otro, ejemplo de purpura a azul. Esto se hace en Css3 mediante las etiquetas siguientes:

#NOMBRE:hover{
background-color: color;
}

Solo debemos añadir una propiedad "hover" y añadir el cambio de color y este cumplira con las normas
